    `Money.java` -- immutable record in `com.prosperity.shared`:
    ```java
    public record Money(BigDecimal amount) {
        public Money {
            Objects.requireNonNull(amount, "amount must not be null");
            if (amount.scale() > 2) {
                throw new IllegalArgumentException("Money precision cannot exceed 2 decimal places");
            }
            amount = amount.setScale(2, RoundingMode.HALF_UP);
        }
        public static Money of(String value) { return new Money(new BigDecimal(value)); }
        public static Money ofCents(long cents) { return new Money(BigDecimal.valueOf(cents, 2)); }
        // NO of(double) method -- floating-point forbidden
        public Money add(Money other) { return new Money(this.amount.add(other.amount)); }
        public Money subtract(Money other) { return new Money(this.amount.subtract(other.amount)); }
        public long toCents() { return amount.movePointRight(2).longValueExact(); }
    }
    ```

    - `convertToDatabaseColumn(Money money)` returns `money.toCents()`
    - `convertToEntityAttribute(Long cents)` returns `Money.ofCents(cents)`
    - Annotate with `@Converter(autoApply = false)` (explicit `@Convert` on entities)
