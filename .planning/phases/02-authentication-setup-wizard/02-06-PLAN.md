  <action>
Create `frontend/src/app/auth/setup.ts` as a standalone Angular component.

**Component structure:**
- Standalone component with inline template and styles (Angular 21 simplified naming)
- Reactive form with 3 fields: email, password, displayName
- PrimeNG components: `p-floatlabel`, `p-inputtext`, `p-password`, `p-button`, `p-message`
- Full-screen centered layout per D-09 and UI-SPEC auth form layout

**Form fields (from UI-SPEC):**
- Email: `p-floatlabel` + `pInputText`, label "Adresse email", validation: required + email format
- Password: `p-floatlabel` + `p-password` with `[toggleMask]="true"` and `[feedback]="true"`, label "Mot de passe"
- Display name: `p-floatlabel` + `pInputText`, label "Nom d'affichage", validation: required + min 2 chars

**Password validation UI (per D-14 and UI-SPEC):**
Real-time feedback below password field as a vertical stack:
- "12 caracteres minimum" — check: `password.length >= 12`
- "1 lettre majuscule" — check: `/[A-Z]/.test(password)`
- "1 chiffre" — check: `/\d/.test(password)`
- "1 caractere special" — check: `/[^a-zA-Z0-9]/.test(password)`
Each row shows `pi-check` icon (green `--p-green-500`) when met, `pi-times` icon (muted `--p-text-muted-color`) when not.

**Submit behavior:**
- CTA button: "Creer le compte" (full-width, primary severity)
- Loading state: button shows spinner + "Creation en cours..." + disabled
- On success: show `p-message` with severity="success" and text "Compte cree avec succes. Vous pouvez maintenant vous connecter." Then after 2 seconds, redirect to `/login` via `router.navigate(['/login'])`
- On 409 error: show `p-message` with severity="error" and text "Le compte administrateur existe deja. Rendez-vous sur la page de connexion."
- On other error: show generic network error

**Validation messages (from UI-SPEC):**
- Email required: "L'adresse email est requise"
- Email invalid: "Format d'email invalide"
- Password required: "Le mot de passe est requis"
- Display name required: "Le nom d'affichage est requis"
- Display name min: "2 caracteres minimum"

**Layout (from UI-SPEC):**
- Page background: `bg-surface-50`
- Card: `max-w-md mx-auto p-8 rounded-xl shadow-md bg-surface-0`
- Centered vertically and horizontally: `min-h-screen flex items-center justify-center`
- Heading: "Bienvenue sur Prosperity" (text-2xl font-semibold)
- Subtext: "Creez le compte administrateur pour demarrer." (text-sm text-muted-color)
- Field gap: `gap-4`
- CTA: full-width, `mt-4`

**Accessibility (from UI-SPEC):**
- Auto-focus email field on load
- `aria-live="polite"` on error/success message container
- Tab order: email > password > display name > submit
