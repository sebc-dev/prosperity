| Role | CSS Variable / Value | Usage |

- "Ajouter une transaction" primary CTA button

- Negative amounts (debits): `--p-red-500` with `-` prefix (default display, amounts stored as signed centimes)

- **Focal point:** "Ajouter une transaction" button -- top right of content area, accent-filled
- Page heading: "Transactions -- {Account Name}" (24px/600), left-aligned. Account name from route param resolution
- Filter bar: `bg-surface-50` background, `p-4` padding, `gap-4` between fields, `rounded-lg` corners. Collapsed by default on mobile, always visible on desktop
- Table: full width, `p-datatable-sm`, lazy server-side pagination
- Amount column: right-aligned, `tabular-nums`, colored by sign (green positive, red negative)
- Pointed column (P): `pi-check` icon when pointed (accent color), empty when not. Click toggles pointed status
- Actions column: `pi-pencil` (edit) + `pi-trash` (delete) icon buttons. Only shown for `source === 'MANUAL'` transactions
- Source badge: small `p-tag` in Description column or as separate column if space permits. "Manuel" for MANUAL, "Recurrent" for RECURRING
- Pagination: bottom of table, shows current page info and rows-per-page selector


- Dialog width: `32rem` (512px)

## Interaction States

### Table Rows

| State | Visual |
|-------|--------|
| Default | White background, normal text |
| Hover | `--p-surface-50` background |
| Manual transaction row | Edit and delete action buttons visible |
| Non-manual transaction row (RECURRING) | Action buttons hidden (read-only in Phase 5) |
| Pointed | `pi-check` icon in Pointed column, accent color |
| Not pointed | Empty Pointed column cell |
| Loading (row action) | Spinner icon replaces action icon on the active row |

### Filter Bar

| State | Visual |
|-------|--------|
| Default | All filter fields empty, showing placeholders |
| Active filters | Filled fields retain values. "Filtrer" button always enabled |
| Loading results | `p-table` shows loading spinner overlay |
| No results | Table body shows empty state message (see Copywriting) |

### Pagination

| State | Visual |
|-------|--------|
| Single page | Paginator hidden (fewer than `rows` results) |
| Multiple pages | Paginator visible with page numbers, prev/next arrows |
| Active page | Accent color highlight on current page number |
| Loading page change | Table loading overlay while fetching new page |

### Dialogs

| State | Visual |
|-------|--------|
| Opening | PrimeNG default fade-in animation |
| Form invalid | "Enregistrer" button disabled (opacity 0.6). Amount is required, date is required |
| Submitting | "Enregistrer" shows spinner + "Enregistrement...", button disabled |
| Error (generic) | `p-message` severity="error" below form fields |
| Error (access denied) | `p-message` severity="error" with "Vous n'avez pas les droits pour cette operation." |

### Pointed Toggle

| State | Visual |
|-------|--------|
| Click to point | `pi-check` icon appears, brief loading spinner during API call |
| Click to unpoint | `pi-check` icon disappears, brief loading spinner during API call |
| API error | Previous state restored, error toast or inline message |

## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Table semantics | `p-table` renders native `<table>` with `<thead>` / `<tbody>`, column headers have `scope="col"` |
| Sort announcement | PrimeNG `p-table` native `aria-sort` on sortable columns (Date, Amount) |
| Dialog focus trap | PrimeNG `p-dialog` implements native focus trap and `role="dialog"` |
| Dialog close | Escape key closes dialog (PrimeNG default) |
| Action buttons | `aria-label` on icon-only buttons: "Modifier la transaction du {date}", "Supprimer la transaction du {date}" |
| Pointed toggle | `aria-label="Pointer la transaction"` / `aria-label="Depointer la transaction"`, `role="checkbox"` with `aria-checked` |
| Error announcement | `aria-live="polite"` on error message containers in dialogs and page |
| Confirmation dialog | `p-confirmdialog` uses `role="alertdialog"` with descriptive `aria-label` |
| Keyboard navigation | Tab order through filter fields, table rows, action buttons. Enter/Space toggles pointed |
| Empty state | Empty state region has `role="status"` so screen readers announce it |
| Amount semantics | Amounts include `aria-label` with full text: "moins 45 euros et 30 centimes" pattern for screen readers |
| Pagination | PrimeNG paginator provides native `aria-label` on page buttons |
| Filter form | Filter section wrapped in `role="search"` with `aria-label="Filtrer les transactions"` |
